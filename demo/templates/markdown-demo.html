**NOTE:** to prevent the browser from stripping out the whitespace, text inside the markdown tab must be included in a `pre` tag. This is removed before rendering the markdown unless you include `allow_pre="true"` as an option to the `markdown` tag. If you pass in content as a tag option then the `pre` tags are not necessary.
